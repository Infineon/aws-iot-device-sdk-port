AWS IoT Device SDK Port Library
Overview

This library is a port layer implementation for the Cypress MQTT and HTTP Client libraries to work with the AWS-IoT-Device-Embedded-C SDK library on Cypress PSoC 6 connectivity enabled platforms. This library APIs are not expected to be called by application directly. Please refer to the MQTT and HTTP Client library documnetation for more details.
